# Nand2Tetris Project

## Overview

This project is part of the **Nand2Tetris course**, which guides the development of a computer system from basic logic gates to a functioning computer capable of running programs.  

The goal of this project is to implement a **software-hardware stack** including:

- **Hardware**: Logic gates, ALU, CPU, and memory.
- **Software**: Jack compiler, assembler, virtual machine, and high-level programs.

By completing this project, I gained practical experience with **computer architecture, logic design, and software implementation** from the ground up.

## Features

- Implementation of basic logic gates using only NAND
- Construction of combinational circuits (Adders, ALU)
- Sequential circuits for memory and CPU control
- Hack computer simulator using the Hack assembly language
- Jack high-level language compilation and execution
- Sample programs demonstrating arithmetic, loops, and conditionals
